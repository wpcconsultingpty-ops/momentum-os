Momentum OS
Momentum OS is a personal self-motivation dashboard designed to help track daily mindset, stress, habits, journaling, and recovery. The current version is a single-file web app with four main views: Dashboard, History, Journal, and Reset Coach.

What it does
The app is built to give a simple daily control panel rather than a complicated wellness system. It combines a daily score, streaks, weekly review, calendar history, journaling, and a guided Reset Coach interface in one place.

Current features
Daily mood and stress sliders.

Six core habit checkboxes.

Daily reflection notes and tomorrow focus field.

Weekly trend bars and 7-day review.

Monthly calendar with selectable saved days.

Journal view with prompts and saved entry display.

Reset Coach view with support-oriented chat UI and safety messaging.
